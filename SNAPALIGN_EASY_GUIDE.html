📦 PART 1: Create Your Database (MongoDB Atlas)

This is like a big box where all your products and orders are saved!

1 Open MongoDB Website

Go to this website in your browser:

https://www.mongodb.com/atlas
🖥️ Open Browser → Type Website → Press Enter
2 Create Free Account

Click the big green button that says "Try Free" or "Start Free"

  • ✍️ Enter your email
  • ✍️ Create a password
  • ✅ Click Sign Up
💡 You can also click "Sign up with Google" if you have Gmail!
3 Create Your Database Box

After logging in:

  • 👆 Click "Build a Database"
  • 🆓 Select "M0 FREE" (the free one with $0)
  • 🌍 Provider: Choose AWS
  • 📍 Region: Choose Mumbai (ap-south-1) - closest to India!
  • 📝 Name it: snapalign-cluster
  • ✅ Click "Create"
🆓 Free Forever = ₹0 Cost!
4 Create a Username & Password

A popup will ask for username and password:

FieldWhat to Type
Usernamesnapalign_user
PasswordClick "Autogenerate" and COPY IT!
⚠️ IMPORTANT: Copy the password and save it somewhere safe! You'll need it later!
5 Allow Everyone to Connect

On the left side, click "Network Access"

  • 👆 Click "Add IP Address"
  • 👆 Click "Allow Access from Anywhere"
  • ✅ Click "Confirm"
🌍 Now your app can talk to the database from anywhere!
6 Get Your Secret Connection Link

On the left side, click "Database"

  • 👆 Click "Connect" button
  • 👆 Click "Connect your application"
  • 📋 Copy the connection string

It looks like this:

mongodb+srv://snapalign_user:<password>@snapalign-cluster.xxxxx.mongodb.net/snapalign
⚠️ Replace <password> with the password you saved in Step 4!
✅ DONE! Your database is ready! Save your connection string!

3 Create New Web Service

  • 👆 Click "Web Service"
  • 🔍 Find your snapalign repository
  • 👆 Click "Connect"
4 Fill in the Settings

Scroll down and click "Advanced" → "Add Environment Variable"

KeyValue

  • ⏳ Wait 5-10 minutes for it to build

Copy your backend URL (looks like this):

✅ DONE! Your backend is online! Save your Render URL!

💳 PART 7: How to Change Payment Keys (Razorpay)

To accept real payments, you need real Razorpay keys!

⚠️ Currently using DEMO MODE - payments are simulated, no real money!
1 Get Razorpay Keys
  • Go to dashboard.razorpay.com
  • Create account or login
  • Go to Settings → API Keys
  • Click "Generate Key"
  • Copy Key ID (starts with rzp_live_ or rzp_test_)
  • Copy Key Secret
💡 Use rzp_test_ keys for testing, rzp_live_ for real payments!
2 Go to Admin Settings
  • Login to Admin Panel
  • 👆 Click "Settings" tab
3 Update Payment Keys
FieldWhat to Enter
Razorpay Key IDYour Key ID (e.g., rzp_live_xxxxx)
Razorpay Key SecretYour Secret Key
  • 👆 Click "Save Settings"
4 Also Update in Render (Important!)

The admin panel change is temporary. For permanent change:

  • Go to render.com
  • 👆 Click your snapalign-backend service
  • 👆 Click "Environment" tab
  • Update these values:
KeyNew Value
RAZORPAY_KEY_IDYour new Key ID
RAZORPAY_KEY_SECRETYour new Secret
  • 👆 Click "Save Changes"
  • ⏳ Wait for redeployment (2-3 mins)
