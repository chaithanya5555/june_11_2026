👤 Users Collection Schema
Purpose: Stores all customer and employee accounts with role-based access control.
📋 users
user_id String Unique identifier (UUID) Required
email String User's email address Required
name String Display name Required
role Enum "customer" | "employee" | "manager" Required
phone String Contact number Optional
address Object Shipping address details Optional
created_at DateTime Account creation timestamp Auto
// Example User Document { "user_id": "usr_a1b2c3d4e5f6", "email": "employee@snapalign.in", "name": "Rahul Kumar", "role": "employee", "phone": "+91 98765 43210", "warehouse_access": true, "created_at": "2025-07-15T10:30:00Z" }

📦 Products Collection Schema
💰 Financial Fields: This collection includes landed cost calculations for profit tracking!
📋 products
product_id String Unique product identifier Required
name String Product display name Required
price Number Selling price in INR (₹) Required
cost_price Number Landed cost in INR Required
alibaba_price_usd Number Original Alibaba price ($) Optional
shipping_cost_usd Number Shipping from China ($) Optional
duty_percent Number Import duty % (default: 18) Optional
compare_at_price Number Original price for discounts Optional
stock Number Available inventory count Required
bin_location String Warehouse location (e.g., "A1") Optional
category String Product category Required
// Example Product with Landed Cost { "product_id": "prod_phone_case_001", "name": "iPhone 15 Pro Max Clear Case", "category": "Phone Cases", // Alibaba Import Costs "alibaba_price_usd": 1.50, "shipping_cost_usd": 0.30, "duty_percent": 18, // Calculated Landed Cost (INR) "cost_price": 177.02, // Auto-calculated // Selling Prices (INR) "price": 499, "compare_at_price": 999, // Inventory "stock": 150, "bin_location": "Shelf A, Bin 3" }
🎟️ Coupons Collection Schema
⚠️ Profit Guardrail: System blocks coupons that would make (Selling Price - Discount) < Landed Cost!
📋 coupons
coupon_id String Unique coupon identifier Required
code String Customer-facing code (e.g., "SAVE20") Required
discount_type Enum "percentage" | "fixed" Required
discount_value Number Amount (% or ₹) Required
min_cart_value Number Minimum order amount (₹) Optional
max_discount Number Maximum discount cap (₹) Optional
valid_from DateTime Start date Required
valid_until DateTime Expiry date Required
usage_limit Number Max total uses Optional
times_used Number Current usage count Auto
is_active Boolean Enable/disable coupon Required
// Example Coupon Document { "coupon_id": "coup_summer2025", "code": "SUMMER25", "discount_type": "percentage", "discount_value": 25, "min_cart_value": 500, "max_discount": 200, "valid_from": "2025-06-01T00:00:00Z", "valid_until": "2025-08-31T23:59:59Z", "usage_limit": 1000, "times_used": 234, "is_active": true }
🚢 Alibaba Shipments Collection Schema
📋 alibaba_shipments
shipment_id String Unique shipment identifier Required
tracking_number String Carrier tracking number Required
supplier_name String Alibaba seller name Required
items Array List of products in shipment Required
total_cost_usd Number Total Alibaba invoice ($) Required
shipping_cost_usd Number Shipping charges ($) Required
duty_paid_inr Number Customs duty paid (₹) Optional
status Enum "ordered" | "shipped" | "customs" | "delivered" Required
order_date DateTime When order was placed Required
expected_delivery DateTime Expected arrival date Optional
// Example Alibaba Shipment { "shipment_id": "ship_2025_07_001", "tracking_number": "SF1234567890", "supplier_name": "Shenzhen Phone Accessories Co.", "items": [ { "product_id": "prod_001", "qty": 500, "unit_price_usd": 1.50 }, { "product_id": "prod_002", "qty": 300, "unit_price_usd": 2.20 } ], "total_cost_usd": 1410.00, "shipping_cost_usd": 180.00, "duty_paid_inr": 25650, "status": "customs", "order_date": "2025-07-01T00:00:00Z", "expected_delivery": "2025-07-20T00:00:00Z" }
⚡ Recommended Indexes
// MongoDB Index Commands // Users - Fast lookup by email and role db.users.createIndex({ "email": 1 }, { unique: true }) db.users.createIndex({ "role": 1 }) // Products - Fast search and category filter db.products.createIndex({ "product_id": 1 }, { unique: true }) db.products.createIndex({ "category": 1 }) db.products.createIndex({ "name": "text" }) // Orders - Fast user lookup and date sorting db.orders.createIndex({ "user_id": 1, "created_at": -1 }) db.orders.createIndex({ "status": 1 }) // Coupons - Fast code lookup db.coupons.createIndex({ "code": 1 }, { unique: true })
