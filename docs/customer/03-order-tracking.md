# How to Track Your Order

## Overview
Use the public order tracking page to check the real-time status of your order. No login required.

---

## Step-by-Step Instructions

### Step 1: Open Tracking Page
Two ways to get there:

**Way 1: Direct URL**
1. Open browser
2. Go to: `https://your-site.com/track`

**Way 2: From Sidebar Menu**
1. Click hamburger menu (☰) in top-left
2. Click **Track Order**

### Step 2: Enter Order ID
1. You'll see a search box: "Enter Order ID"
2. Type your order ID exactly as received
   - Format: `ORD-XXXXXXXX` (e.g., `ORD-A1B2C3D4`)
   - You received this on the payment success page and in your order history
3. Click the blue **Track** button

### Step 3: View Order Status

#### Progress Bar
A visual 4-step progress bar shows:
1. **Placed** (🕐) — Order was placed
2. **Confirmed** (✅) — Payment verified
3. **Shipped** (🚚) — Package dispatched
4. **Delivered** (🏠) — Package delivered

- Active steps are highlighted in blue
- Remaining steps are grayed out
- The progress bar fills proportionally

#### Order Details
Below the progress bar:
- **Order ID** — Your unique reference
- **Date** — When order was placed
- **Total** — Amount paid (₹)
- **Tracking Number** — Courier tracking # (if added by admin)
- **Items** — List of products ordered

#### If Order is Cancelled/Expired
- Progress bar shows in red
- Message: "Order cancelled" or "Order expired"

### Step 4: Check Tracking Number
If the admin has added a tracking number:
1. You'll see a blue highlighted box: **Tracking Number**
2. Copy this number
3. Use it on the courier's website:
   - **Delhivery**: [delhivery.com/track](https://www.delhivery.com/track)
   - **BlueDart**: [bluedart.com/tracking](https://www.bluedart.com/tracking)
   - **India Post**: [indiapost.gov.in](https://www.indiapost.gov.in/VAS/Pages/trackconsignment.aspx)

## Where to Find Your Order ID

| Source | How to Find |
|--------|-------------|
| **Payment Success Page** | Shown right after payment |
| **My Orders Page** | Login → Profile → My Orders → Each order shows ID |
| **Email** | If email notifications are set up |

---

## Troubleshooting

| Problem | Solution |
|---------|----------|
| "Order not found" | Double-check the Order ID, it's case-sensitive |
| No tracking number | Admin hasn't added it yet. Check back later |
| Status still "Pending" | Payment may not have completed. Contact support |

---

## Time Required
- Track an order: ~10 seconds
