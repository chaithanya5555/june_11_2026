# How to Use WhatsApp Customer Support

## Overview
Snap Aligner provides instant customer support via WhatsApp. A floating green button is always visible.

---

## Using the WhatsApp Button

### Step 1: Find the Button
1. Look at the **bottom-right corner** of any page
2. You'll see a green circular button with the WhatsApp logo
3. It's always visible, even when scrolling

### Step 2: Click to Chat
1. Click the green **WhatsApp button**
2. WhatsApp opens (app or web version):
   - **Mobile**: Opens WhatsApp app directly
   - **Desktop**: Opens WhatsApp Web in a new tab
3. A pre-written message appears: "Hi SnapAlign, I need help"
4. You can edit the message or send as-is

### Step 3: Send Your Query
1. Type your question or concern
2. Hit Send
3. The support team will respond during business hours

---

## Common Support Queries

| Topic | Example Message |
|-------|----------------|
| Order status | "Hi, I'd like to check the status of order ORD-A1B2C3D4" |
| Product inquiry | "Do you have tempered glass for Samsung S24?" |
| Return/refund | "I want to return my order ORD-XXXX, the case doesn't fit" |
| Bulk order | "I need 50 units of the braided USB-C cable for my office" |
| Tracking issue | "My tracking number isn't working: DELY12345" |

## Changing the WhatsApp Number (For Admin)
1. Open file: `/app/frontend/src/components/Navbar.js`
2. Find the WhatsApp link: `href="https://wa.me/919999999999..."`
3. Replace `919999999999` with your actual WhatsApp number (with country code, no +)
4. Example: For +91 98765 43210, use `919876543210`
