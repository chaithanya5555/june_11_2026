🔌 Part D: Linking Frontend to Backend API
🧒 Simple Explanation: The frontend (Vercel) needs to know where to find the backend (Render). It's like giving your store the address of your warehouse!
1
Verify Backend URL

Your Render backend URL format:

https://snapalign-backend.onrender.com

Test it works by visiting:

https://snapalign-backend.onrender.com/api/health

You should see:

{"status": "healthy", "database": "connected"}
2
Update Vercel Environment Variable
  • Go to Vercel Dashboard
  • 👆 Click your project
  • 👆 Click "Settings" → "Environment Variables"
  • Find REACT_APP_BACKEND_URL
  • Update value to your Render URL
  • 👆 Click "Save"
3
Redeploy Frontend
  • Go to "Deployments" tab
  • 👆 Click "..." on latest deployment
  • 👆 Click "Redeploy"
  • ⏳ Wait 2 minutes
✅ Frontend is now connected to Backend!
