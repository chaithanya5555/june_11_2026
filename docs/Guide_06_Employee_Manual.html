📦 Order Workflow

Follow these steps for every order:

1️⃣ Pending → 2️⃣ Picked → 3️⃣ Packed → 4️⃣ Shipped
📍
Step 1: Pick Items
  • ☐
    Go to the bin location shown
  • ☐
    Pick the items listed
  • ☐
    Check quantity is correct
  • ☐
    Tap "Items Picked"
📦
Step 2: Pack Order
  • ☐
    Get correct size box/mailer
  • ☐
    Add bubble wrap if fragile
  • ☐
    Include packing slip
  • ☐
    Seal the package
  • ☐
    Tap "Packed"
🏷️
Step 3: Add Shipping Label
  • ☐
    Print shipping label
  • ☐
    Stick label on package
  • ☐
    Scan tracking barcode
  • ☐
    Tap "Ready to Ship"
✅

Order Complete!

Great job! Move to next order.

❓ Common Problems
📷 Camera won't open?
  • Check if you allowed camera permission
  • Go to Settings → Apps → Browser → Permissions → Camera → Allow
📦 Item not in bin?
  • Check nearby bins
  • Tap "Report Issue" → "Item Missing"
  • Manager will be notified
🔴 Barcode won't scan?
  • Clean the barcode
  • Try better lighting
  • Tap "Enter Manually" and type order number
📱 App is slow?
  • Check WiFi connection
  • Close other apps
  • Refresh the page
