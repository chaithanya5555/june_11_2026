💰 Part 4: Monitoring INR Margins
1
View Profit Dashboard

Go to: Dashboard → Financial Summary

You'll see:

MetricWhat It Means
💵 Gross RevenueTotal sales before any deductions
📦 Landed Cost (COGS)What you paid for the products
💳 Payment Fees (2%)Razorpay charges
🎟️ Coupon DiscountsMoney lost to coupons
📈 Net ProfitYour actual profit!
📊
Financial Summary Table
2
Check Product Margins

Go to: Products → Click any product → View Margin

For each product, you can see:

  • 💰 Selling Price: ₹499
  • 📦 Landed Cost: ₹209
  • 📊 Margin: ₹290 (58%)
⚠️ Warning Colors:
🟢 Green (>40%) = Good margin
🟡 Yellow (20-40%) = Okay margin
🔴 Red (<20%) = Low margin - consider raising price!
3
Review Coupon Impact

Go to: Dashboard → Coupon Analytics

See how much money coupons are costing you:

  • 📊 Total coupon usage this month
  • 💸 Total discount given
  • 📈 Orders with vs without coupons
💡 Tip: If a coupon is causing losses, the system will automatically BLOCK it!

⚙️ Part 6: Changing Settings
1
Change Admin Password

Go to: Settings → Security → Change Password

  • ✍️ Enter current password
  • ✍️ Enter new password (min 8 characters)
  • ✍️ Confirm new password
  • 👆 Click Save
⚠️ Security Tip: Use a strong password with letters, numbers, and symbols!
2
Update Razorpay Keys

Go to: Settings → Payment → Razorpay

FieldWhere to Get It
Key IDRazorpay Dashboard → Settings → API Keys
Key SecretRazorpay Dashboard → Settings → API Keys
  • ✍️ Paste your new Key ID
  • ✍️ Paste your new Key Secret
  • 👆 Click Save Settings
✅ Test Mode vs Live Mode:
• rzp_test_xxx = Test mode (no real money)
• rzp_live_xxx = Live mode (real money!)
3
Update Exchange Rate

Go to: Settings → Financial → Exchange Rate

  • ✍️ Enter current USD to INR rate (e.g., 83.50)
  • 👆 Click Update
💡 Check xe.com for current exchange rates!
