📐 Guide 4: Financial Logic & Math

Landed Cost Calculator & Profit Guardrail System

🧒 Simple Explanation: When you buy stuff from China (Alibaba), you pay for the product + shipping + taxes. We need to know the TOTAL cost before we can decide the selling price. This guide shows you the math to make sure you never lose money! 💰
📦 Part 1: Landed Cost Calculator

The "Landed Cost" is the TOTAL cost of getting a product from Alibaba to your warehouse in India.

THE MAGIC FORMULA
Landed Cost = (Alibaba Price + Shipping) × (1 + Duty%) × 1.18 × Exchange Rate

The 1.18 multiplier accounts for 18% GST

Formula Components:

Component What It Means Example
Alibaba Price Cost per unit on Alibaba (USD) $1.50
Shipping Shipping cost per unit (USD) $0.30
Duty % Import duty rate (varies by product) 18%
GST Goods & Services Tax (fixed 18%) ×1.18
Exchange Rate Current USD to INR rate ₹83.50

💻 Python Code Implementation
# Landed Cost Calculator Function def calculate_landed_cost( alibaba_price_usd: float, shipping_cost_usd: float, duty_percent: float = 18.0, gst_percent: float = 18.0, exchange_rate: float = 83.50 ) -> float: """ Calculate the total landed cost in INR. Args: alibaba_price_usd: Unit price from Alibaba ($) shipping_cost_usd: Shipping per unit ($) duty_percent: Import duty rate (default 18%) gst_percent: GST rate (default 18%) exchange_rate: USD to INR rate Returns: Landed cost in INR (₹) """ # Step 1: FOB Cost (Free on Board) fob_cost = alibaba_price_usd + shipping_cost_usd # Step 2: Add Import Duty after_duty = fob_cost * (1 + duty_percent / 100) # Step 3: Add GST after_gst = after_duty * (1 + gst_percent / 100) # Step 4: Convert to INR landed_cost_inr = after_gst * exchange_rate return round(landed_cost_inr, 2) # Example Usage cost = calculate_landed_cost( alibaba_price_usd=1.50, shipping_cost_usd=0.30, duty_percent=18 ) print(f"Landed Cost: ₹{cost}") # Output: ₹209.25
🛡️ Part 2: Profit Guardrail System
⚠️ THE GOLDEN RULE:

(Selling Price - Coupon Discount) must be > Landed Cost

If this rule is violated = BLOCKED! 🚫
PROFIT GUARDRAIL CHECK
Net Price = Selling Price - Coupon Discount

IF Net Price < Landed Cost → 🚫 BLOCK ACTION

What Gets Blocked?

Action When Blocked
Creating new coupon If any product would result in loss
Applying coupon to order If order total after discount < total landed cost
Setting product price If new price < landed cost

💻 Profit Guardrail Code
# Profit Guardrail Validation class ProfitGuardrailError(Exception): """Raised when an action would result in a loss.""" pass def validate_coupon_profitability( selling_price: float, landed_cost: float, discount_amount: float ) -> dict: """ Check if a coupon would result in profit or loss. Returns: dict with 'is_valid', 'net_price', 'profit', 'message' """ net_price = selling_price - discount_amount profit = net_price - landed_cost if net_price < landed_cost: return { "is_valid": False, "net_price": net_price, "profit": profit, "message": f"🚫 BLOCKED: Would lose ₹{abs(profit):.2f} per sale!" } return { "is_valid": True, "net_price": net_price, "profit": profit, "message": f"✅ ALLOWED: Profit ₹{profit:.2f} per sale" } # API Endpoint Example @api_router.post("/coupons/validate") async def validate_coupon(request: Request): body = await request.json() product = await db.products.find_one( {"product_id": body["product_id"]} ) result = validate_coupon_profitability( selling_price=product["price"], landed_cost=product["cost_price"], discount_amount=body["discount_amount"] ) if not result["is_valid"]: raise HTTPException( status_code=400, detail=result["message"] ) return result
📋 Quick Reference Card
Formula Use Case
FOB = Price + Shipping Basic cost before taxes
After Duty = FOB × (1 + Duty%) Add import duty
After GST = After Duty × 1.18 Add 18% GST
Landed = After GST × Exchange Rate Convert to INR
Min Price = Landed Cost + Buffer Never sell below this!
Profit = Selling - Landed - Fees Your actual profit
💡 Pro Tip: Always add a 10-20% buffer above landed cost for payment fees, returns, and unexpected costs!
