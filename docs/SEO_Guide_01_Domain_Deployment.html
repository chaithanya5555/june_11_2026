🗺️ Auto-Sitemap Generator Script

Backend endpoint that auto-generates sitemap when products are added:

# Add to server.py - Auto-sitemap generator @api_router.get("/sitemap.xml", response_class=Response) async def generate_sitemap(): """Auto-generate SEO sitemap with all products""" products = await db.products.find({}, {"_id": 0}).to_list(1000) base_url = "https://snapalign.in" # Static pages static_pages = [ ("/", "1.0", "daily"), ("/shop", "1.0", "daily"), ("/collections/iphone-17-tempered-glass", "0.9", "weekly"), ("/collections/samsung-s26-screen-protector", "0.9", "weekly"), ("/collections/pixel-10-tempered-glass", "0.9", "weekly"), ("/about", "0.5", "monthly"), ("/contact", "0.5", "monthly"), ] xml = '<?xml version="1.0" encoding="UTF-8"?>\n' xml += '<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">\n' # Add static pages for path, priority, freq in static_pages: xml += f''' <url> <loc>{base_url}{path}</loc> <changefreq>{freq}</changefreq> <priority>{priority}</priority> </url>\n''' # Add product pages dynamically for product in products: slug = product.get("slug") or generate_seo_slug(product["name"]) xml += f''' <url> <loc>{base_url}/products/{slug}</loc> <changefreq>weekly</changefreq> <priority>0.8</priority> </url>\n''' xml += '</urlset>' return Response(content=xml, media_type="application/xml") def generate_seo_slug(name: str) -> str: """Generate SEO-friendly slug from product name""" import re slug = name.lower() slug = re.sub(r'[^a-z0-9\s-]', '', slug) slug = re.sub(r'[\s_]+', '-', slug) return slug.strip('-')

📋 Migration Checklist: Netlify → Vercel
1 Export Domain from Netlify

Netlify → Domain settings → Remove domain from site (keep DNS)

2 Import to Vercel

Vercel → Add New Project → Import from GitHub → Select snapalign repo

3 Configure Build Settings
  • Root Directory: frontend
  • Build Command: yarn build
  • Output Directory: build
4 Set Environment Variable

REACT_APP_BACKEND_URL = https://snapalign-backend.onrender.com

5 Add Custom Domain

Vercel → Settings → Domains → Add snapalign.in

6 Update DNS Records
TypeNameValue
A@76.76.21.21
CNAMEwwwcname.vercel-dns.com
✅ Done! Your site is now on Vercel with auto-sitemap generation!
